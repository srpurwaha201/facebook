# Facebook - Clone using Ruby on Rails
This is a **Facebook** clone with the following features implemented : 

- Create an account with email confirmation

- Create a post with image

- Comment, edit, like on posts

- Send Friend Request to other Users

- Custom feed for individual users

- Search for other users using search bar with autocomplete feature

- Custom profile page for each user

- Create Post on other user's page

**Note :** 

- All the above features are implemented using Ajax.
- The complete UI of the app is inspired from Facebook
